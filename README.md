# Bandana Users, Wrapped

A static, single page informational deck summarizing three years (2023 to 2026) of Bandana user research: what jobseekers told us and what to build next.

## Note on data and privacy
The **raw research data is intentionally excluded** from this repo. Source material, including Slack exports, meeting transcripts, private notes, the interviews directory, and the methodology page, is kept out of the deploy because it contains personal and internal information. Interviewees in the deck are referenced only by initials. Do not add the raw data folders here if this repo is public.
